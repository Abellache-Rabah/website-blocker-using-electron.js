<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* node js
  ```sh
  apt install nodejs
  ```

### Installation



1. Clone the repo
   ```sh
   gh repo clone rabah01abellache/website-blocker-using-electron.js
   ```
   
2. cd to the folder
   ```sh
   cd website-blocker-using-electron.js/
   ```
   
3. Install NPM packages
   ```sh
   npm install
   ```
4. run the start script 
   ```js
   npm start
   ```
